## 🎯 AI Agent 使用流程

```
1. 阅读 ai_agent_prompt.md
   ↓
2. 判断模型复杂度
   ├─ 简单（1-2部件）→ 直接 GNodesBuilder
   └─ 复杂（3+部件）→ 多流构建模式
   ↓
3. 理解"API使用优先级"
   ↓
4. 编写代码时查阅 api_quick_reference.md
   ↓
5. 遵循"禁止模式"
```

---

## ⚠️ 重要原则

AI Agent在使用这些文档时，务必遵循：

1. **复杂模型**：使用多流构建模式（独立部件 → `merge_objects()` 合并）
2. **优先级**：组合模板 > 语义化API > 手动API
3. **禁止**：手动计算 `atan2()` 再 `set_rotation()`
4. **检查**：是否有现成的组合模板可用
5. **复杂度提升**：善用 `G_Instance_On_Points` 和 `G_Array_*` 节点
